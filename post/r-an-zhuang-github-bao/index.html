R安装Github包

2020-02-24

##一、代码安装：
######1.R安装devtools包

install.packages("devtools")
library(devtools)

######2.安装github上的R包（需翻墙或改hosts）

devtools::install_github('lchiffon/REmap')
#q前为github的用户名，后为包名

######3.测试

library(REmap)

##二、本地安装
在github.com上直接下载xx-master.zip,将zip文件解压成文件夹XX-master，输入命令

install.packages("D:/nCov2019-master",repos=NULL,type="source")
